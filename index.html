Parte I – Introducción – HTML, CSS y JavaScript

Introducción

HyperText Markup Language (HTML), Cascading Style Sheets (CSS), y JavaScript son los principales lenguajes que hacen que la web funcione.

Estos lenguajes están estrechamente relacionados, pero están diseñados para tareas muy específicas. Vamos a comprender cómo interactúan durante esta sesión.

HTML es para agregar significado a tu contenido, dandole marcado (markup).

CSS es para darle formato (estilo) al contenido marcado.

JavaScript es para darle interacción al formato y el contenido marcado.

¿Cómo funciona cada uno?

HTML

Puedes marcar una ejecución particular de texto como un párrafo con este HTML:

Este es un párrafo.

CSS

Luego, puedes especificar el tamaño y color del párrafo con algo de CSS:

p {
    font-size: 20px;
    color: blue;
}

JavaScript

Y si nos queremos poner exquisitos, podemos re-escribir el párrafo cuando el usuario clickea, con algo de JavaScript

var p = document.getElementById('parrafo');

    p.addEventListener('click', function(event) {
      p.innerHTML = 'Hiciste click en este párrafo.';
    });

Conclusión

HTML, CSS y JavaScript son lenguajes totalmente diferentes.

La mayoría de los sitios web dependen de los tres, pero la apariencia de cada sitio está determinada por HTML y CSS.

Parte I – Creando una página web básica - Setup

Setup / Preparación

Comencemos creando una nueva, llamemosla 'pagina-web-basica'. Luego, dentro de la carpeta, crearemos un nuevo archivo HTML llamado basico.html.

Este archivo HTML representa una sola página web, y es donde pondremos todo nuestro código.

El flujo de trabajo básico de los desarrolladores web es editar HTML en el editor de texto y luego ver los cambios en un navegador web, por lo que es exactamente lo que debe hacer para cada sección de este capítulo.
Parte I – Creando una página web básica - Estructura de una página web

Estructura de una página web

Agreguemos el siguiente código HTML a nuestro archivo basico.html.

<!DOCTYPE html>
    <html lang="es">
        <head>
        <!-- Metadata va acá -->
        </head>
        <body>
        <!-- tu increíble contenido va acá -->
        </body>
    </html>

Veamos que és cada parte

Primero, necesitamos decir a los navegadores que se trata de una página web HTML5 con la línea :

<!DOCTYPE html>

Esta es sólo una cadena especial que los navegadores buscan cuando intentan mostrar nuestra página web.

Declaración de tipo de documento en HTML 4.01

<!DOCTYPE html PUBLIC 
        "-//W3C//DTD XHTML 1.0 Transitional//EN" 
        "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

Declaración de tipo de documento en HTML 5



                        <!DOCTYPE html>
