Parte I – Section I – HTML, CSS y JavaScript

Introducción

HyperText Markup Language (HTML), Cascading Style Sheets (CSS), y JavaScript son los principales lenguajes que hacen que la web funcione.

Estos lenguajes están estrechamente relacionados, pero están diseñados para tareas muy específicas. Vamos a comprender cómo interactúan durante esta sesión.

HTML es para agregar significado a tu contenido, dandole marcado (markup).

CSS es para darle formato (estilo) al contenido marcado.

JavaScript es para darle interacción al formato y el contenido marcado.

¿Cómo funciona cada uno?

HTML

Puedes marcar una ejecución particular de texto como un párrafo con este HTML:

Este es un párrafo.

CSS

Luego, puedes especificar el tamaño y color del párrafo con algo de CSS:

p {
    font-size: 20px;
    color: blue;
}

JavaScript

Y si nos queremos poner exquisitos, podemos re-escribir el párrafo cuando el usuario clickea, con algo de JavaScript

var p = document.getElementById('parrafo');

    p.addEventListener('click', function(event) {
      p.innerHTML = 'Hiciste click en este párrafo.';
    });

Conclusión

HTML, CSS y JavaScript son lenguajes totalmente diferentes.

La mayoría de los sitios web dependen de los tres, pero la apariencia de cada sitio está determinada por HTML y CSS.

Parte I – Sección III - Metodología y herramientas

Metodología y herramientas

Modo aprendizaje: activado

La mejor manera de aprender a desarrollar es a través de la construcción de algo que te interese.

A medida que desarrolles habilidades, comprenderás lo que es posible y eso te inspirará a crear nuevos proyectos.

Al aplicar tus habilidades en un proyecto interesante, podrás disfrutar más del proceso de aprendizaje y aprender a un ritmo más rápido.

Consejo: comienza a trabajar en un propio proyecto tan pronto como puedas, mientras vas a ir ganando más habilidades, siguiendo éstas sesiones.
